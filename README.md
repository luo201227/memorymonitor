# memorymonitor
Swing常被以执行效率慢和内存占用过高等问题受到攻击<br/>
其实简单直接的说：Swing效率没有问题<br/>
此内存监控工具条它的作用是监测目前Java的总的申请的堆内存以及使用的内存<br/>
